# Numerical Analysis Project

## Project Overview
This project contains implementations and reports of various Numerical Analysis methods used for solving equations, interpolation, regression, differentiation, integration, and ordinary differential equations (ODEs).  
The goal of this project is to study and compare different numerical techniques and understand their accuracy and convergence behavior.

---

## Implemented Methods

### Root Finding Methods
- Bisection Method  
- Newton Raphson Method  
- Halley Method  
- False Position Method  
- Secant Method  
- Fixed Point Iteration Method  
- Double Root Aitken Process Method with Newton Raphson Method  

---

### Linear System Methods
- Jacobi Method  
- Gauss-Seidel Method  

---

### Non-Linear Equation Methods
- Jacobi Method for Non-Linear Equations  

---

### Interpolation Methods
- Lagrange Interpolation Method  
- Newton Divided Difference Interpolation Method  
- Newton Forward Interpolation Method  
- Newton Backward Interpolation Method  

---

### Regression
- Regression Method  

---

### Numerical Integration Methods
- Trapezoidal Method  
- Simpson Method  

---

### Numerical Differentiation Methods
- Two Points Formula Method  
- Three Points Formula Method

### Ordinary Differential Equation (ODE) Methods

- Modified Euler Method
